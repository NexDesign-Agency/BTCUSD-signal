# 💹 BTC/USD Pro-Trading Signal Console

A professional-grade real-time trading dashboard for BTC/USD using Binance WebSocket API. Built with Vite, Lightweight Charts, and Price Action Logic.

## 🚀 Features
- **Zero-Lag Logic**: Instant trend detection based on EMA Cross and Price Action Scoring.
- **Market Structure Recognition**: Automated detection of HH/HL (Uptrend) and LH/LL (Downtrend).
- **Pro Pending Orders**: Automatic calculation of Entry, SL, and TP (1, 2, 3) based on H4 ATR.
- **Visual Confidence Meter**: real-time confluence scoring (0-100%).
- **Multi-Timeframe Matrix**: Live status for M5, M15, H1, and H4.
- **Interactive Controls**: Auto-fit chart, individual indicator toggles (MA, BB, VOL, MACD).
- **Smart Alerts**: 
  - Voice assisted signals ("Signal Buy!").
  - Audio "Ding" for priority transitions.
  - Browser Push Notifications.

## 🛠️ Tech Stack
- **Frontend**: Vanilla JS (Vite)
- **Charting**: Lightweight Charts (TradingView)
- **Math Engine**: TechnicalIndicators (npm)
- **Data Source**: Binance WebSocket API

## 📦 Installation & Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/NexDesign-Agency/BTCUSD-signal.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
